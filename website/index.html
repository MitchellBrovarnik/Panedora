Live Synced Lyrics

Time-coded lyrics auto-scroll with the music, highlighted line by line. Choose from glow, pill, or full-line highlight styles.

Audio Visualizers

10 animated background effects powered by Web Audio API. Reactive bars, circles, and waves that respond to your music in real time.

Always-on-Top Mini Player

A compact floating bar that stays above all windows — even borderless fullscreen games. Controls and feedback at a glance.

Adaptive Dynamic Theme

Samples your current album art and extracts a dominant color, repainting the entire UI in real time as songs change.

Private and Secure

Credentials encrypted with OS keychain. No third-party tracking. All data stored locally and wiped on sign-out.

Built with a modern Electron stack.

Main Process

Orchestrates the app lifecycle, manages all API requests from a secure Node.js context, and controls Mini Player window state via IPC.

API Controller

Handles all communication with Pandora's backend REST endpoints, routing through Chromium's native network stack with managed auth tokens.

Renderer Process

Vanilla JS, HTML5, CSS3 frontend. Handles routing, audio playback, the full theme system, and the Adaptive Color extraction algorithm.

Web Audio Visualizer

Standalone class powered by the Web Audio API with three Canvas-based reactive styles and configurable FFT sizes with math smoothing.

Secure Context Bridge

IPC bridge via Electron's contextBridge with full context isolation — the renderer has zero access to Node.js or Electron APIs.

Dynamic CSS System

Built entirely on CSS custom properties. Employs CSS Grid, Flexbox, backdrop-filter, and a transparent frameless window setup.

About this project

Panedora is a personal, educational, and experimental project created purely for fun. It was built as a creative exercise to explore modern UI design (Glassmorphism) and Electron-based desktop application development. This project is intended for personal use only and was developed with zero intent to cause harm, bypass security, or interfere with Pandora's business operations.

A Note on the Backend Code

You are free to explore and adapt the frontend Glassmorphism UI code for your own non-commercial projects. However, to prevent abuse, the backend logic has been intentionally obfuscated in public releases.
